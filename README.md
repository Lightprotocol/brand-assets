## Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Light Blue | `#0066FF` | Primary brand color |
| Black | `#111111` | Dark backgrounds, text |
| White | `#FFFFFF` | Light backgrounds, reversed logos |

---

## Usage

- Use SVG for web and digital. Use PNG when SVG is not supported.
- Don't modify, rotate, distort, or recolor the logos.
- Don't place logos on busy backgrounds without sufficient contrast.
- Respect the safety zone.
